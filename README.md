# GoodFutures Landing Page

AI-driven careers marketplace landing page built with React and Material-UI.

## Quick Start

```bash
npm install
npm run dev
# Visit http://localhost:5173
```

## Tech Stack

- React 18 + Vite
- Material-UI (MUI) v5
- Google Fonts (Marcellus, Inter)

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run preview` - Preview production build

## Design System

**Colors**
- Primary: `#1E3B33` (Forest Green)
- Secondary: `#FFB800` (Amber/Gold)
- Backgrounds: `#ECECEA` (Candidates), `#6B8B7E` (Companies)

**Typography**
- Headings: Marcellus (serif)
- Body: Inter (sans-serif)
